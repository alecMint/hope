
--- Post Theme Install ---

Add following line to [theme]/functions.php
require_once(TEMPLATEPATH.'/../../../assets/functions.php');


Symlink for style.css needs to be created
cd [path to theme]
rm style.css
ln -s ../../uploads/x/style.css style.css


May need to add the stylesheet href to header.php or mod-document.head.php
<link rel="stylesheet" type="text/css" href="<?php bloginfo('stylesheet_url'); ?>" />


Add assets to [theme]/template-parts/mod-document-head.php
<link rel="stylesheet" type="text/css" href="<?php vres('/assets/ace.css'); ?>" />
<script src="<?php vres('/assets/ace.js'); ?>"></script>


May need to upgrade jquery version
http://ajax.googleapis.com/ajax/libs/jquery/1.8/jquery.min.js
